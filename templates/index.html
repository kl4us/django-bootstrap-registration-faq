This is a basic Django project with some existing apps included. A discovered pinax (and his basic projects) few months ago but i don't like because it isn't simple for a new (noob like me) django developer. This project includes the following apps:

  • django-registration (for user registration)
  • django-bootstrap-toolkit (for bootstrap forms)
  • django-debug-toolbar (for local debug)

I also added a simple Faq app. For this app is required django-summernote to have a simple WYSIWYG editor based on Twitter's Bootstrap.
